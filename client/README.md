# Plan It - Wall Calendar

A beautiful, production-quality interactive wall calendar built with Next.js, TypeScript, and Tailwind CSS.

## Quick Start

```bash
cd client
npm install
npm run dev
```

Open http://localhost:3000

## Features
- Wall Calendar Layout with hero images and spiral binding
- Violet Dusk theme (light/dark mode)
- Date range selection with notes
- Indian public holidays display
- Keyboard navigation
- SessionStorage persistence for notes

## Tech Stack
- Next.js 16 + TypeScript
- Tailwind CSS v4
- Lucide React icons
